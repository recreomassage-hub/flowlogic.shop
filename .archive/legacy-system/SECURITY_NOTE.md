# ⚠️ Security Note

**Важно:** Этот архив был очищен от секретов перед коммитом в git.

## Очищенные данные

- ✅ GitHub токены (`ghp_*`) → заменены на `ghp_REDACTED`
- ✅ AWS ключи (`AKIA*`) → заменены на `AKIA_REDACTED`
- ✅ Stripe ключи → заменены на `*_REDACTED`

## Причина

Git hooks проверяют коммиты на наличие секретов. Все реальные токены и ключи были удалены из архива для безопасности.

## Восстановление

Если нужно восстановить функциональность скриптов из архива:

1. **GitHub токены:** Используйте GitHub Secrets или переменные окружения
2. **AWS ключи:** Используйте AWS IAM роли или переменные окружения
3. **Stripe ключи:** Используйте переменные окружения

**Никогда не коммитьте реальные секреты в git!**
